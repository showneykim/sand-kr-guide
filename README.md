# 샌드: 레이더스 오브 소피 — 한국어 입문 가이드

**SAND: Raiders of Sophie**([Steam](https://store.steampowered.com/app/1431300/SAND_Raiders_of_Sophie/))의 **비공식 한국어 입문 가이드** 레포입니다. 게임이 공식 한국어를 지원하지 않아, 새 플레이어가 **설치부터 첫 추출(extraction)까지** 막힘없이 진행하도록 돕는 것이 목표입니다.

## 📖 가이드 보기

- **메인 가이드 페이지(비주얼): [`index.html`](index.html)** — 다크 디젤펑크 테마, 주석 스크린샷 + 컨셉 다이어그램.
  - 로컬에서 보려면 `index.html`을 브라우저로 열면 됩니다.
  - *(GitHub Pages로 게시하면 `https://showneykim.github.io/sand-kr-guide/` 에서 바로 열람 — 현재는 로컬 작업 중)*

## ℹ️ 게임 한 줄 소개

거대 보행 기계 **"트램플러"** 를 조립해 사막 행성 **소피**의 폐허를 약탈하고, 전리품을 들고 **살아서 탈출(추출)** 하는 **솔로~6인 PvPvE 추출 슈터**. 대체역사 1910년 오스트리아-헝가리 제국 디젤펑크 세계관.

## 🛠️ 제작 메모

- 출처: 사용자 인게임 스크린샷(1차) + 다출처 웹조사 + 적대적 교차검증 + 4렌즈 검토.
- 페이지는 **Jekyll을 쓰지 않습니다**(`.nojekyll`). 순수 HTML/CSS + 인라인 SVG.
- 한글 웹폰트 Pretendard(CDN), 폴백 Malgun Gothic.
