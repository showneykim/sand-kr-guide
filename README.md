## ℹ️ 게임 한 줄 소개

거대 보행 기계 **"트램플러"** 를 조립해 사막 행성 **소피**의 폐허를 약탈하고, **모래폭풍이 닫히기 전에 전리품을 들고 탈출**하는 **솔로~6인 PvPvE 추출 슈터**. 대체역사 1910년 오스트리아-헝가리 제국 디젤펑크 세계관.
