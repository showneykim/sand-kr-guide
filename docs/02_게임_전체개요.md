## 0. 한 줄 요약

거대 **보행 전쟁기계 "트램플러(Trampler)"** 를 직접 조립해, 모래로 뒤덮인 행성 **소피(Sophie)** 의 폐허를 약탈하고, **모래폭풍이 닫히기 전에 전리품을 들고 탈출(추출)** 하는 **솔로~6인 협동 PvPvE 추출(extraction) 슈터**. ✅

대체역사 1910년·오스트리아-헝가리 제국 **디젤펑크** 세계관. 우크라이나 스튜디오 제작, tinyBuild 배급. ✅

## 2. 세계관 · 스토리

스토리 모드는 없고, **환경 스토리텔링 + 팩션 설정**으로 세계관을 전달한다. ✅

- **시대/미학**: 대체역사 **1910년경**. 인류가 새 에너지원을 발견해 우주 진출 → **오스트리아-헝가리 제국**이 행성 정복("우주 식민 경쟁")을 주도. 미학은 **디젤펑크**(스팀펑크 아님). ✅
  - ⚠ 주의: "디젤 동력"은 **미학 표현일 뿐**, 트램플러의 실제 동력은 **리액터 + 에너지 로드**다(디젤 엔진 아님). ✅(교차검증 정정)
- **소피(Sophie)**: 한때 제국의 풍요로운 식민지 행성. **원인불명의 생태 재앙으로 바다가 증발** → 가라앉았던 해저·섬·난파선·폐허 도시가 드러난 **사막 세계**가 됨. 식민지인들은 황급히 탈출. (절차적 생성 맵) ✅
- **레이더스(플레이어)**: 카이저가 소피의 땅을 약속했던 **갈리치아(Galicia, 우크라이나인·폴란드인 다수 거주의 실제 제국 영토) 출신 생존자/후손**들. 재앙 후 "역사적 부당함을 바로잡고 정당한 몫을 되찾겠다"며 **반(半)불법으로 행성에 귀환**해 약탈. (대학교수·괴짜 연구자·오컬티스트·기술자·퇴역군인 등도 포함) ✅
  - → 우크라이나 개발사의 문화적 색채. 풀 우크라이나어 현지화 제공, 러시아 시장 미출시.
- **위협 — Upior(우피르)**: 폐허 도시·기념물에 출몰하는 **초자연적 언데드/괴물 PvE 적**. 슬라브·폴란드 민속의 흡혈 망령 'upiór'(vampire의 어원)에서 유래. **장갑 트램플러엔 위협이 적지만, 도보 플레이어를 떼로 덮쳐 죽인다.** 그 외 AI 적 트램플러('Ironclad'로 불림 ◐), 모래벌레(sandworm)도 존재. ✅
- **세 팩션(테크트리/상점 분기)** — 모두 오스트리아-헝가리 역사 오마주: ✅
  - **Godlewski's Expedition**(고들레프스키 원정대): 차대·모터리액터·엔진·승무원실·유틸리티·메드킷.
  - **K.K. Landwehr**(K.K. 란트베어, 실제 제국 향토방위군): 장갑·무기·조향·돌파장비·개량탄.
  - **Kaiser's Friends**(카이저의 친구들): 대포(오토캐논/샷건캐논)·화물 보관·갑판/발코니.
- **화폐 "Crowns(크라운)"**: 실제 오스트리아-헝가리 화폐 'krone'에 대한 오마주. ✅

## 3. 핵심 게임 루프

게임은 두 공간을 오간다(스크린샷 상단 탭): ✅
- **ORBIT(궤도/모선)** — 출격 준비. 청사진·창고·상점·테크트리·트램플러 에디터.
- **EXPEDITION(원정)** — 실제 사막에서의 플레이.

루프: **모선에서 트램플러 제작·연구 → 원정 출격 → 약탈 → 폭풍 닫히기 전 무전탑/구조선으로 탈출 → 전리품으로 더 강하게 성장 → 반복.** ✅

### 두 가지 모드 ✅
| | **Voyage(항해)** | **Storm Dive(스톰 다이브)** |
|---|---|---|
| 성격 | 지속형 오픈월드, 시간 제한 없음 | 세션 기반 **배틀로얄식** |
| 위험/보상 | 저위험·저티어, 여유로운 탐험·연습 | 고위험·고티어, 강제 PvP |
| 추출 | 무전탑(탈출 지점) 상시 활성 | **모래폭풍 단계 진행** 후 활성화 |
| 종료 | 원할 때 추출 | 폭풍이 맵 중심(거대 난파선 **Dreadnought**)으로 수축(약 2시간 ◐) |
| 추천 | **초보자는 여기서 시작** | 익숙해진 뒤 도전 |

**모드 선택 화면(실물 UI) ✅(스크린샷 확정)**: 모드 카드 옆 **RIVALS(상대 종류)** 를 함께 고른다 — **Solo only / Small Crews / Large Crews**. 솔로 입문자는 **Solo only** 선택 시 솔로끼리 매칭돼 한결 공평. (Voyage 원문: "Unlimited match duration / Evacuation zones are always active"; Storm Dive 원문: "Match duration is limited / gradually covered by a sandstorm".)

> **중요**: "오프라인 레이드"(로그아웃 중 약탈당함)는 없음 — 트램플러가 세션 종료 시 함께 추출되므로 접속 안 했을 때 잃지 않음. ✅

## 4. 트램플러(Trampler) — 게임의 심장

거대 모듈식 보행 기계 = **이동 수단 + 본거지 + 전리품 저장고 + 전투 플랫폼 + 리스폰 거점**. 깊은 에디터로 정찰형~중장갑 요새까지 자작 가능. ✅

### 필수 구성 (에디터 Requirements) ✅(스크린샷 확정)
**섀시(Chassis) 1개 + 핵심 구획 4개**:
1. **Chassis(섀시/차대)** — 기반 골격. 크기(Small/Middling/Great/Royal ◐)에 따라 무게·수용량 결정.
2. **Steering(조향)** — 조타. 여기 **조종 콘솔(Control Panel)** 로 운전. 없으면 못 움직임.
3. **Motor-Reactor(모터-리액터)** — **가장 핵심**. 동력원. 적의 1순위 표적. 에너지 로드로 작동.
4. **Captain compartment(선장실)** — 선장 침대(리스폰) + **나포 장치**가 있는 방.
5. **Entrance(입구)** — 사다리로 올라올 수 있는 유효 탑승 경로 필요.

### 동력 ✅
- **리액터 + 에너지 로드(Energy Rod, NZ MK2)**. 새 로드 1개당 **약 10~12분 주행** ◐. 신규 캐릭터는 **약 5개**(자료에 따라 3개 ⚠)로 시작.
- **F 키**로 로드 장전. **리액터를 켜면 검은 연기**가 나서 다른 플레이어에게 위치 노출 → 약탈·정차 중엔 끄는 게 정석. ◐
- 무게↑ = 기동성↓. 엔진은 무게를 보상하지만 에너지를 더 먹음.

### 블루프린트 & 테크트리 ✅
- **Blueprint(설계도)**: 저장된 디자인. 파괴돼도 재건 가능. **계정 내 모든 캐릭터·지역이 공유**(반대로 인벤토리·테크트리 진행은 **지역별 고정**). ✅
- 프리메이드 예시: **Nervous Bug / Dusty Sentinel / Stoic Pilgrim**(스크린샷). 초반 추천 **Grumpy Walker** = 인게임 확정 ✅(스탯 예: 기동성 49 · Crew 3 · Cannons 2 · Cargo 1 · 제작비 ≈ 295크라운).
- **테크트리**: 세 팩션 × **4티어**(약 95개 노드 ◐). 노드 해금 비용 = **크라운 + 재료**. **해금은 레시피만 열고, 실제 제작엔 재료가 또 든다.** 티어I ~600~1,750 / 티어IV 9,000~12,000 크라운 + 희귀재료(Black Box·Crystal·Ficus 등) ◐⚠.

## 5. 원정 플레이

### 출격 준비 흐름 ✅(스크린샷 확정)
출격 준비는 상단 **3단계 탭**으로: **I. GEAR UP(장비 챙기기) → II. LOAD THE TRAMPLER(트램플러에 싣기) → III. OVERVIEW(점검) → READY(출격)**. OVERVIEW에서 보급 슬롯·트램플러 스탯을 확인 후 출격.

### 출발 전 준비(반드시) ✅
**아무것도 장전된 채로 시작하지 않는다.** 출격 직후 매번:
1. **대포 장착 + 포탄 장전**(구경 일치: 40mm 대포엔 40mm 포탄). 장비는 선반의 녹색 박스에 담겨 스폰.
2. **에너지 로드** 리액터에 장전 → **리액터 가동(F)**.
3. **냉장고에 통조림(Canned Food)** 채우기 → 사망 시 트램플러 침대에서 **빠른 리스폰**(없으면 ~1분 대기 또는 팀 추출까지 대기 ◐).
4. **개인 무기 2정 + 각 전용 탄약**(예: 리볼버 근중거리 + 소총 장거리). 무방비로 POI 진입 = 사실상 사망. ◐

### 약탈 & 인벤토리 ✅
- 개인 주머니는 **매우 작다**. 월드에 널린 **빈 녹색 박스**를 손에 들고 다니며 전리품을 담아 트램플러 선반·화물칸에 보관.
- **추출로 인정되는 위치 = 개인 인벤토리 / 손에 든 박스 / 트램플러 보관(선반·랙·화물칸)**. **갑판 바닥에 흩어진 물건은 분실.** ✅
- **잠긴 상자(Locked Box)**: 작은 열쇠(슬롯당 6개 적재)로 **추출 전에 반드시 열어야** 인정. 열쇠만 들고 추출하면 안 됨. ◐
- 전리품 등급: Common→Uncommon→Rare→Noteworthy→Experimental. **Metal Scrap(고철)** 이 만능 건축 재료. ◐

### 전투 — 개인 & 트램플러 vs 트램플러 ✅
- **개인 화기**: 무기마다 **전용 탄약**. 잘못된 탄은 발사 불가.
- **대포 3종**: **40mm 오토캐논**(연사·발열식) / **80mm 함포**(대형 단발·장거리, 매발 재장전) / **70mm 샷건 캐논**(근거리). 모두 빈 채로 스폰 → 직접 장전. ✅
- **적 트램플러 공략(스크린샷 Combat Manual + 가이드)**:
  - **다리(Legs)** 먼저 → 기동 불능(다리당 속도 감소). → **대포** 무력화로 반격 차단 → **리액터** 파괴 = **가장 빠른 격침**.
  - 화물칸 파괴=보급 방해 / 승무원실 파괴=리스폰 지연 / 문 파괴=내부 진입 / 조향 파괴=정지.
- **수리**: 경미 손상(새는 파이프·회로·금속 패치) = **멀티툴(T)**. 대형 손상(파괴된 구획·벽) = **Repair Kit**(충전 3회, 사용 전 **리액터 OFF** 필요). 격침된 트램플러 구획은 복구 불가(대포 제외). ✅
- **나포(Seize/Boarding)**: 적 트램플러에 승선 → **선장실의 마스터 컨트롤 장치**를 길게 상호작용 → 함선 탈취(EA에선 영구 소유 불가). 적의 **리스폰 거점을 무력화**(다음 사망 시 매치 아웃). 선장실이 파괴되면 불가. ✅
  - ⚠ 스팀 포럼 초보 불만 1위가 "boarding meta"(내 배에 침입당해 캡틴실 장악) → **내부·선장실 방어 중요**.

## 6. 폭풍 · 추출(Extraction)

### 모래폭풍(Storm Dive) ✅
- 시간이 지나면 **폭풍 원이 수축**. 안에 갇힌 트램플러는 피해, 완전히 닫히면 플레이어도 피해.
- **탈출 지점은 폭풍 단계 진행 후 활성화**(Phase 2 종료 시 지도 표시). 최종 탈출 지점은 끝에서 두 번째 단계 종료 시 공개.
- **최종 원(Final Circle)**: 폭풍이 모래를 휘저어 **추가 고급 전리품**을 드러냄 — 위험 감수 보상.

### 추출 절차(전리품 보관 유일 수단) ✅
1. 지도의 **흰 '문' 아이콘 = 탈출 구역**으로 트램플러를 몰고 가 평지에 정차(완전 종료 말고 'Halt' 권장 ◐).
2. **무전탑**에 올라 라디오 조작 → **"Call For Evacuation"** → **약 90초 타이머** 시작.
3. 트램플러에서 **초록 연기**가 솟음 = 주변 모든 플레이어에게 위치 광고(→ 미리 망원경으로 정찰). ✅
4. **취소 조건**: 트램플러를 운전 / 적 트램플러 포격을 맞음 / 적이 라디오 조작 → 처음부터 재시작.
5. 90초 후 구조선 도착 → **케이블/짚라인 투하(약 60초 유지)** → 탑승해 상승.
- **트램플러가 파괴돼도 도보 추출 가능** — 단, **개인 인벤토리 + 손에 든 박스 1개**만 챙김(나머지 전부 소실). ✅

## 7. 경제 · 진행

- **화폐 = Crowns(크라운) 단일**. 추출 시 귀중품(샤르도네·문서·교역품 등)이 자동 환전, 나머지는 상점 판매. **Black Box ≈ 500 크라운**, 라디오 비콘 상자 ≈ 2,000 크라운. ✅
  - **Black Box**: 테크트리 상위 노드 재료로 필수. **적 트램플러 격침 시 리액터 밑에 드롭**(주 획득처는 PvP). 다 쓴 뒤엔 개당 ~500에 판매. ◐
- **상점(Shop)**: 모선 허브의 **창고(Storage)** 메뉴/터미널. **추출 성공 후에만** 이용. **플레이어 간 거래 없음.** 스페이스바 길게 눌러 확정 ◐. (탄약·메드킷·열쇠 실수로 팔지 않게 주의)
- **크래프팅**: 트램플러 워크숍/요새 벤치에서 탄약·무기·자원 변환. 고급 레시피는 티어2 워크숍 필요. ◐
- **메타 진행**: 전통적 **캐릭터 레벨/XP 없음**. 성장 = 트램플러+블루프린트+크라운+연구. 테크트리·창고는 **지역 고정**, 블루프린트만 계정 공유. ✅

## 11. 초보자 핵심 팁 (가이드북 1장 초안 후보)

1. **Voyage 모드부터.** 폭풍 압박 없이 맵·추출·트램플러 조작을 익혀라(첫 10~15시간). 그다음 Storm Dive. ◐
2. **출격 = 매번 정비.** 대포 장전(구경 일치)·에너지 로드·냉장고 통조림. 아무것도 장전된 채 시작 안 함. ✅
3. **약탈 땐 리액터 OFF.** 검은 연기로 위치 노출되고 연료도 아낌. ◐
4. **빈 녹색 박스를 들고 다녀라.** 개인 주머니는 너무 작다. ✅
5. **전투: 다리 → 리액터.** 다리로 묶고, 리액터로 끝낸다. 피격 즉시 멀티툴로 수리. ◐
6. **욕심 금지 — 일찍 추출.** "한 곳만 더"가 전멸의 순간. 목표 달성하면 떠나라. 솔로 손실 대부분이 이미 이득 본 뒤 발생. ◐
7. **초반엔 중형(Medium) 트램플러.** 느린 중장갑은 포위당함. 부수면 싸게 재건되는 균형 빌드(예: Grumpy Walker ◐). 자원의 40% 이상 한 대에 몰빵 금지, 예비 1~2대 ◐.
8. **솔로 설계**: 조타·리액터·대포를 가까이 배치해 혼자 다 관리. ◐
9. **추출 = 코어 스킬.** PvP 승리보다 "전리품 들고 살아 나오기"를 먼저 익혀라. ✅
10. **돈벌이**: 금고·색상 상자 약탈 + 귀중품/Black Box 판매. Black Box는 적 트램플러 격침으로만 안정 획득. ◐
