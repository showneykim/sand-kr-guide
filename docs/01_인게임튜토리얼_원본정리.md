# SAND: Raiders of Sophie — 인게임 튜토리얼 원본 정리 (1차 자료)

> 출처: 사용자가 직접 캡처한 인게임 튜토리얼 카드 13장 (`screenshots-source/`).
> 게임 내 영어 원문을 한국어로 옮기고 해설을 덧붙임. **가장 신뢰도 높은 1차 자료.**
> 캡처일: 2026-06-26.

게임 메뉴는 상단에 두 개의 큰 탭이 있다:
- **ORBIT (궤도/모선)** — 출격 준비 공간. 청사진/창고/테크트리/트램플러 에디터 등이 여기 있음. (단축키 Q 쪽)
- **EXPEDITION (원정)** — 실제 사막에 내려가서 하는 플레이. (단축키 E 쪽)

각 튜토리얼 카드 하단 진행 표시(◀ A — — — — — D ▶)는 A/D 키로 페이지를 넘긴다는 의미.

## [타이틀 화면] 서버 선택
- 게임명: **SAND — RAIDERS OF SOPHIE**
- 개발/배급 로고: **tinyBuild(배급), HOLOGRYPH, TOWERHAUS(개발)**
- 하단: "Powered by Microsoft Azure PlayFab", "Communications by VIVOX"
  → **온라인 멀티플레이 전용에 가까운 구조** (전용 서버 + 보이스챗).
- 서버 리전 목록: West US / **East US(선택됨, 2ms)** / Central US / Great Britain / Europe / Asia / South America / Oceania / **South Korea** 등.
  → **South Korea 리전이 존재**하므로 한국에서 핑은 괜찮게 플레이 가능.

## [ORBIT > EXPEDITION] 크루 / 초대
- **Tutorial**: 튜토리얼은 현재 게임 상태에 따라 바뀜. 게임 곳곳에서 `?` 아이콘이나 **F1** 키로 도움말을 열 수 있음.
- **Invite Friends**: 프렌드 리스트에서 초대. **크루(crew)는 최대 6명**.
- **Use Invite Code**: 초대 코드로 합류가 더 빠르고 안전. 코드 공유로 검색/친구추가 없이 바로 합류. ("My Crew 1/6", "Code: XWL735…" 표시).
  → **솔로~6인 협동** 플레이. 코드 기반 파티.

## [ORBIT] BLUEPRINTS (청사진)
- **Manage Blueprints**: 청사진으로 트램플러를 제작. 미리 만들어진(premade) 청사진을 고르거나 직접 설계 가능.
  - 예시 프리메이드 청사진 이름: **Nervous Bug / Dusty Sentinel / Stoic Pilgrim** (각각 필요 자원·승무원 수 표기, 예: 4,000 크라운 등).
- **Build Trampler**: 청사진 선택 후 "Select". **필요한 구획(compartments)과 충분한 자원**이 있어야 제작 가능.
  → 트램플러 = 모듈(구획)을 조립해 만드는 거대 보행 기계.

---

## [ORBIT] STORAGE (창고)
- **Inventory**: 캐릭터 창고. 원정에서 얻은 모든 아이템·자원이 보관됨. (화면상 Est. value 9,999,999 = 보유 자산 가치)
- **Shop**: 상점. **crowns(크라운, 화폐)** 와 자원을 사용해 창고의 물건을 사고팖.
  → 화폐 단위 = **크라운**. 상점에서 무기/탄약/자원 등 거래.

---

## [ORBIT] TECH TREE (테크트리)
- **Gather Resources**: Sophie(무대가 되는 세계) 곳곳에 흩어진 재료·자원 수집. "뭐가 필요할지 모르니 모아둬라."
- **Unlock Technologies**: 보유 자원으로 **팩션(faction) 연구 노드**를 해금. 각 노드를 해금하면 하나 이상의 **새 구획**에 접근 가능.
  - 예시 노드: `IV(a). Royal Chassis (9,000)`, `III(b). Steering (6,000)`, `IV(a). Enclosed Artillery Compartment (9,000)` — 로마숫자 단계 + 비용 구조.
- **Upgrade Your Trampler**: 새로 해금한 구획을 에디터에서 써서 트램플러의 화력·성능 강화.
  → 진행 = 자원 모아 → 연구로 구획 해금 → 더 강한 트램플러 제작. **팩션별 연구 트리** 존재.

## [EXPEDITION] EXPEDITION PLANNING (원정 준비물)
원정 전에 트램플러에 실어야 할 것:
- **Canned Food (통조림)**: 음식은 **리스폰 시간 단축**에 중요. 트램플러 냉장고/화물칸에 실린 음식 1개당 빠른 리스폰 1회.
- **Weapons & Ammo**: 적과 싸우려면 총 몇 자루 + 호환 탄약을 챙길 것.
- **Cannon Shells (대포 포탄)**: 트램플러에 단 **대포 1문당 최소 한 묶음**의 포탄.
- **Ammo 호환표**: 무기마다 쓰는 탄약이 다름(권총/소총/산탄총 등 → 각기 다른 탄약). "L" 표시 = 큰 아이템은 선반 한 칸을 통째로 차지.
- **Recommended Setup (첫 솔로 원정 권장)**: 통조림 10, 권총 2, 탄약 100, 무기 2, 대포포탄 40, (대형무기 L) 1, 탄약 40 — 대략 이런 구성 예시.

## [EXPEDITION] TRAMPLER EDITOR (트램플러 에디터)
- **Movement(편집 조작)**:
  - **R / F**: 트램플러 층(floor) 전환 (다층 구조).
  - **G**: 위층 구획 숨기기 (아래층 편집용).
  - **T**: 문(door) 연결 편집.
  - **Q / E**: 카메라 회전.
  - **Tab**: Compartment Library(구획 라이브러리) 열기.
- **Specifications(성능)**:
  - **Maneuverability(기동성)**: 가감속·회전·험지 주행 능력.
  - **Engines(엔진)**: 무게를 보상하지만 에너지를 소모 → **리액터에 에너지 로드(energy rods) 공급** 필요.
  - **Weight(무게)**: 무거울수록 기동성 하락.
- **Requirements(필수 구획)**: 샤시 1개 + **핵심 구획 4개**를 반드시 설치:
  1. **Chassis(샤시/차대)**
  2. **Steering(조향)**
  3. **Motor-Reactor(모터-리액터)**
  4. **Captain compartment(선장실)**
  5. **Entrance(입구)**
  → (분류상 Chassis 1 + Essentials 4)

## [EXPEDITION] BEGINNING OF THE JOURNEY (출발 절차)
1. **음식 보관**: 냉장고 선반/화물칸 랙에 음식을 넣어 **빠른 리스폰** 확보.
2. **대포 장착**: 대포 거치대에 대포를 설치하고 **포탄 장전**.
3. **리액터 가동**: "Toggle Reactor → Turn On"으로 트램플러에 전원 공급.
4. **정찰**: **B(쌍안경)** 또는 **M(지도)** 로 경로 계획.
5. **조종**: **Control Panel(조종대)** 로 트램플러 운전. "좋은 여행을!"

---

## [EXPEDITION] DURING THE EXPEDITION (원정 중 흐름)
1. 처음엔 **작은 관심지점(points of interest)** 위주로 털어서 무기·장비를 충분히 모은다.
2. 충분해지면 **섬(island)** 에 도착해 탐험·약탈 → 전리품을 **화물칸(cargo)** 에 싣는다.
3. 트램플러를 **E + Evacuation Point(탈출 지점)** 로 몰고 가 **구조선 호출**.
4. 얻은 자원으로 **더 강한 트램플러**를 제작 (루프 반복).
→ 핵심 루프: 작은 곳부터 → 섬 약탈 → 탈출 → 성장.

## [EXPEDITION] AT THE HELM (조종대 조작)
- **Trampler Controls(이동)**: W 전진 / S 후진 / A 좌회전 / D 우회전 / Q 좌측이동 / E 우측이동 (전후진·횡이동·회전 동시 입력 가능).
- **Other Actions**: Lights(조명), Horn(경적), Emergency Brake(비상정지), Take a Closer Look(자세히 보기).
- **Tips**:
  - 조작은 **누르고 있을 필요 없음** — 원하는 값을 설정하면 계속 유지됨.
  - **조타륜 앞에 계속 있을 필요 없음** — 값 맞춰놓고 다른 일 하러 가도 됨.
  → 즉, 자동주행처럼 값만 세팅하고 선원은 수리·전투를 병행.

## [EXPEDITION] COMBAT MANUAL (전투 교본) — 적 트램플러 공략
적 트램플러의 구획별 파괴 효과:
- **Reactor(리액터)** 파괴 → 트램플러 폭발(격침).
- **Steering(조향)** 파괴 → 트램플러 정지.
- **Cannon(대포)** 파괴 → 적의 반격 차단.
- **Cargo(화물칸)** 파괴 → 적의 보급(재보급) 방해.
- **Crew(승무원실)** 파괴 → 적 리스폰 지연.
- **Doors(문)** 파괴 → 트램플러 내부 진입.
- **Legs(다리)** 파괴 → 일시적으로 비틀거림(stagger), 다리 하나당 이동속도 감소.
- **나포(Seize)**: **선장실 안의 장치**를 길게 상호작용하면 적 트램플러를 탈취. 나포 후 적은 그 트램플러에서 리스폰 불가, 우리 팀이 문을 열 수 있음. (단, 선장실이 파괴되면 불가)
- **수리**: 대부분의 구획은 **Failure Point(고장 지점: 압력 파이프/회로 박스/금속 패치)** 를 **멀티툴(T)** 로 고치면 복구 가능. (단, 격침된 트램플러의 구획은 복구 불가 — 대포 제외)

## [EXPEDITION] INVENTORY MANAGEMENT (인벤토리/박스 운반)
- 주머니 용량은 적다. Sophie 곳곳에 **빈 박스(empty boxes)** 가 널려 있음.
- 손에 **박스를 든 채** 아이템을 집으면 박스 안에 들어가고, 넘치면 자동으로 인벤토리에 추가.
- **Tab** 으로 대상 컨테이너에서 아이템 선택/이동.
- 인벤토리 ↔ 손에 든 박스 사이로 아이템 관리(Tab).
  → 전리품은 박스에 담아 운반하는 게 기본. 탈출 시 박스/선반에 정리해야 함.

## [EXPEDITION] STORM DIVE (폭풍 — 배틀로얄식 수축)
- **Storms**: 시간이 지나면 **폭풍 원이 수축**. 폭풍 안에 갇힌 트램플러는 피해를 입고, 완전히 닫히면 플레이어에게도 피해.
- **Evacuation**: 폭풍이 특정 단계에 도달하면(Storm Phase Progress Bar 표시) **탈출 지점이 해금**. **최종 탈출 지점**은 끝에서 두 번째 단계 종료 시 공개.
- **Final Circle**: 마지막 단계엔 폭풍이 모래를 휘저어 **추가 전리품**을 드러냄. 위험을 감수하면 더 값진 전리품 획득 기회.
  → 배틀로얄의 "자기장 수축" + 추출(extraction) 결합.

---

## [EXPEDITION] EVACUATION (탈출 상세)
- **Evacuation Points**: Storm Dive에서 **Phase 2 종료 시** 탈출 지점이 활성화·지도 표시.
- **Evacuation Radius**: 트램플러가 **무선탑(radio tower) 근처**에 있어야 구조선이 픽업 가능.
- **Radio Tower**: 무선탑 위/근처의 라디오로 탈출 호출. (보통 근처에 **흰 연막탄**이 피어 있음)
- **Ship(구조선)**: 보정이 끝나면 구조선 앞쪽에서 **구조 로프**가 내려옴.
- **중요 규칙**:
  - 전리품은 **선반 위 / 손에 들거나 / 인벤토리 안**에 있어야 탈출 시 회수됨. **갑판에 흩어진 미고정 아이템은 분실**.
  - **잠긴 박스(locked boxes)** 는 **오스트리아-헝가리 제국**의 재산으로 보상 없이 몰수됨. (← 세계관 설정: 오스트리아-헝가리 제국)

## 스크린샷에서 확정된 핵심 요약
- **장르**: 멀티플레이(솔로~6인) **PvPvE 추출(extraction) + 배틀로얄식 폭풍 수축** 게임.
- **무대**: "Sophie"라는 사막 세계. 세계관 배경에 **오스트리아-헝가리 제국**.
- **탈것**: **Trampler(트램플러)** — 구획을 조립해 만드는 거대 보행 기계(다층, 리액터/엔진/조향/대포/화물 등).
- **루프**: 모선(ORBIT)에서 트램플러 제작·연구 → 원정(EXPEDITION) 출격 → 약탈 → 폭풍 수축 전에 무선탑/구조선으로 탈출 → 전리품으로 성장.
- **화폐**: crowns(크라운). **테크트리 = 팩션 연구**로 새 구획 해금.
- **전투**: 적 트램플러의 구획별 파괴/나포, 멀티툴 수리, 음식으로 리스폰 단축.
- **한국 관련**: South Korea 서버 리전 존재(핑 양호). 단, UI는 영어.
