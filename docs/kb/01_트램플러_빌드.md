## Trampler 제작 (Trampler Building)

> **범위 주의**: SAND: Raiders of Sophie는 2026-06-22 Steam 앞서 해보기(Early Access) 출시 직후 상태입니다. 아래 **수치·비용·이름은 대부분 EA 패치로 변동될 수 있으며**, 상당수가 AI 생성 SEO 위키에서 나온 값입니다. 신뢰도 태그를 인라인으로 표기했습니다: `[공식]` = Steam/tinyBuild FAQ, `[언론]` = 이름 있는 매체(TheGamer/GameRant 등), `[커뮤니티]` = 실제 플레이어, `[AI위키-미검증]` = 자동 생성 SEO 위키(교차검증한 경우 명시).

### 1. 개요 — 모듈식 제작 시스템

- Trampler(트램플러)는 **"완전히 커스터마이즈 가능한 거대한 보행 메카/요새"** 로, "당신의 창의성과 성공 의지만이 유일한 한계인 모듈형 트램플러를 설계"한다고 공식 설명. `[공식]` (Steam 상점 페이지)
- 게임에는 **"깊고 포괄적인 Trampler Editor(트램플러 에디터)"** 가 있어, 빈 상태에서 칸(compartment)을 하나씩 쌓아 처음부터 제작. `[공식]`
- **청사진(Blueprint)은 캐릭터 간 공유**된다 — 한 번 저장한 설계는 계정 내 모든 캐릭터/모든 서버에서 사용 가능. `[공식]` (tinyBuild FAQ)
- **최대 크루(승무원) 6명**. 솔로 또는 2~6인 팀 플레이 가능하며, **매치메이킹은 크루 규모 기준**. 6인 팀이면 그만큼 크루룸을 충분히 지어야 함. `[공식]`
- 게임 시작 시 **크루 규모별 3개의 기본 청사진**(소형/중형/대형)이 주어짐. 기본 청사진은 기초 부품만 들어 있어 PvP에 약하므로 커스터마이즈 권장. `[언론][커뮤니티]`
- 제작 방식: 블루프린트 메뉴 열기 → 새 설계 생성(`C` 키) → 섀시 배치 → 필수 코어 모듈 부착 → 데크(deck)로 연결 → 전투/유틸 부품 추가 → 저장 후 배치. `[언론]` (nerdschalk)

### 3. 필수 칸(Essential Compartments) — 5종

에디터 우측 하단에 코어 요구사항 툴팁이 떠서 필수 부품 충족 여부를 보여줌. 다음 5개가 없으면 트램플러가 작동하지 않음. `[언론]` (TheGamer, nerdschalk 일치)

1. **Chassis(섀시)** — 모래 위를 걷는 본체/토대.
2. **Motor Reactor(모터 리액터)** — "트램플러에서 가장 중요한 부품". 모든 전자장치·포탑·문에 동력 공급.
3. **Steering / Fly Bridge(조향/플라이 브리지)** — 방향 조종.
4. **Valid Entrance Area(유효 출입구)** — 바닥에서 사다리로 올라올 수 있는 유효 경로 필요.
5. **Captain's Compartment(선장실)** — 트램플러 조종권을 주는 메인 콘솔이 있는 칸.
- 칸들은 **데크(deck)** 로 서로 연결한다. `[언론]`

### 6. 리액터 & 에너지 로드(Energy Rod) 메커니즘

- **Motor-Reactor** 가 트램플러 전체에 동력 공급. **PvP 1순위 표적** — 내부 장갑/보강 칸으로 보호하라. `[언론][AI위키]`
- **연료 = Energy Rod**. 신규 캐릭터는 **NZ Mk2 Energy Rod 5개**를 들고 시작. `[언론]` (TheGamer, GameRant 일치)
- **로드 1개 지속시간: 약 10~12분 연속 주행** 후 리액터 정지. `[언론][AI위키]`
- 실사용 권장: **레이드 1회당 보통 2~4개**면 충분. `[언론]`
- 사용법: 저장 상자에서 여분 로드를 들고 → **메인 스위치(밸브) 옆 전용 슬롯**에 삽입 → **`F` 키**로 충전. 슬롯 위 **연료 게이지**가 빨간색으로 떨어지지 않게 관리. `[언론]`
- **전원 스위치는 굴뚝(chimney) 아래 메인 데크**에 위치. 정차해 약탈할 때 수동으로 전원 OFF → 내부 배터리 절약 + 굴뚝 연기(다른 레이더에게 시각적 신호) 차단. `[AI위키]`
